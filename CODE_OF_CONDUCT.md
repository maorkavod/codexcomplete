# Code of Conduct

This project follows a simple standard: be respectful, constructive, and kind.

## Expected Behavior

- Be welcoming and inclusive.
- Assume good intent.
- Give actionable, respectful feedback.
- Focus on what is best for the project and community.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insulting, or demeaning comments.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove or reject contributions and discussions that do not align with this code of conduct.
