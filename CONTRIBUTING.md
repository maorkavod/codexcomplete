# Contributing

Thanks for your interest in contributing to CodexComplete.

## Development Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Build:
   ```bash
   npm run compile
   ```
3. Open in VS Code and run the extension in an Extension Development Host.

## Pull Request Guidelines

- Keep changes scoped and focused.
- Update docs when behavior or settings change.
- Ensure `npm run compile` passes before opening a PR.
- Add or update tests when introducing non-trivial behavior.

## Reporting Issues

Please include:

- VS Code version
- Extension version
- Repro steps
- Expected vs actual behavior
- Relevant logs/screenshots
